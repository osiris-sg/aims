AIMS Field-Tech PWA — End-to-End Flow

Role-locked NFC scanning, first-tap binding, service report & DO acknowledgement, signature capture.
Start / End Page Decision API call Error / Guard 
flowchart TD
  %% =========================================
  %% Entry & Lockout
  %% =========================================
  Start([User opens AIMS]):::startend --> SignIn[Clerk sign-in]:::page
  SignIn --> Guard[/portal/* loads
FieldOnlyGuard mounts/]:::page
  Guard --> RolesApi[/"GET /users/me/roles"/]:::api
  RolesApi --> OnlyFT{Only
field-tech
role?}:::decision
  OnlyFT -- No --> Portal([Normal portal access]):::startend
  OnlyFT -- Yes --> RedirScan[router.replace /scan]:::page

  %% =========================================
  %% Scan Landing
  %% =========================================
  RedirScan --> ScanLanding["/scan
NFC landing page"]:::page
  ScanLanding --> FlagCheck{enableField
ScanApp
!== false?}:::decision
  FlagCheck -- No --> Blocked[Feature disabled message]:::error
  FlagCheck -- Yes --> NfcSupport{NDEFReader
available?}:::decision

  NfcSupport -- No --> ManualEntry[Show manual SKU input]:::page
  NfcSupport -- Yes --> TapBtn[Show 'Start scanning' button]:::page

  %% Manual fallback path
  ManualEntry --> SkuType[User types SKU]:::page
  SkuType --> SkuApi[/"GET /assets/skuKey/:sku"/]:::api
  SkuApi --> SkuFound{Found?}:::decision
  SkuFound -- No --> SkuErr[Error: No asset with that SKU]:::error
  SkuErr --> ManualEntry
  SkuFound -- Yes --> Chooser

  %% NFC path
  TapBtn --> NfcTap[User taps NFC tag]:::page
  NfcTap --> ReadUid[Read chip UID
event.serialNumber]:::page
  ReadUid --> UidApi[/"GET /assets/by-nfc-uid/:uid"/]:::api
  UidApi --> UidStatus{Status?}:::decision
  UidStatus -- 200 --> Chooser
  UidStatus -- 404 --> BindPage

  %% =========================================
  %% First-tap Binding
  %% =========================================
  BindPage["/scan/bind?uid=...
First-tap bind page"]:::page
  BindPage --> EnterSku[User types SKU printed on item]:::page
  EnterSku --> FindApi[/"GET /assets/skuKey/:sku"/]:::api
  FindApi --> FindOk{Found?}:::decision
  FindOk -- No --> BindErr[Error message]:::error
  BindErr --> BindPage
  FindOk -- Yes --> Confirm[Show asset card
'Link this tag to this item']:::page
  Confirm --> BindApi[/"POST /assets/:id/bind-nfc-tag
body: { uid }"/]:::api
  BindApi --> BindStatus{Status?}:::decision
  BindStatus -- 409 conflict --> BindConflict[Tag already bound to
another asset]:::error
  BindConflict --> BindPage
  BindStatus -- 200 --> Chooser

  %% =========================================
  %% Action Chooser
  %% =========================================
  Chooser["/scan/asset/:assetId
Action chooser"]:::page
  Chooser --> CtxApi[/"GET /maintenance-reports
/scan-context/:assetId"/]:::api
  CtxApi --> ShowAsset[Render asset header
+ recent activity]:::page
  ShowAsset --> HasDO{Latest DO
on file?}:::decision
  HasDO -- Yes --> DoCard[Card: Acknowledge latest DO
enabled]:::page
  HasDO -- No --> DoCardOff[Card: Acknowledge DO
disabled]:::page
  ShowAsset --> ServiceCard[Card: Maintenance
service report]:::page

  %% =========================================
  %% Service Report Branch
  %% =========================================
  ServiceCard --> ServiceNew["/scan/asset/:id/service/new"]:::page
  ServiceNew --> DescIn[User enters description]:::page
  DescIn --> AddPhotos[Tap 'Add photos'
camera or gallery]:::page
  AddPhotos --> UploadApi[/"POST /uploads/image
per photo, returns S3 key"/]:::api
  UploadApi --> PhotosOk[Photos displayed
removable thumbnails]:::page
  PhotosOk --> ContSign1[Tap 'Continue to signature']:::page
  ContSign1 --> CreateApi[/"POST /maintenance-reports
creates draft, status=draft"/]:::api
  CreateApi --> SignPage

  %% =========================================
  %% DO Acknowledgement Branch
  %% =========================================
  DoCard --> DoAck["/scan/asset/:id/do/:doId"]:::page
  DoAck --> NotesIn[User adds optional notes]:::page
  NotesIn --> DoPhotos[Add proof-of-delivery photos]:::page
  DoPhotos --> UploadApi2[/"POST /uploads/image
per photo"/]:::api
  UploadApi2 --> ContSign2[Tap 'Continue to signature']:::page
  ContSign2 --> CreateApi2[/"POST /maintenance-reports
desc prefixed 'DO Ack (DO :id)'"/]:::api
  CreateApi2 --> SignPage

  %% =========================================
  %% Signature
  %% =========================================
  SignPage["/scan/asset/:id/sign?reportId=..."]:::page
  SignPage --> NameOpt[Optional recipient name]:::page
  NameOpt --> DrawSig[Draw on SignatureCanvas
react-signature-canvas]:::page
  DrawSig --> SubmitSig[Tap 'Submit']:::page
  SubmitSig --> SignApi[/"POST /maintenance-reports/:id/sign
body: { signature, signedByName }"/]:::api
  SignApi --> SignOk{Success?}:::decision
  SignOk -- No --> SignErr[Error alert,
stays on sign page]:::error
  SignErr --> DrawSig
  SignOk -- Yes --> Done

  %% =========================================
  %% Done
  %% =========================================
  Done["/scan/asset/:id/done
Success screen"]:::page
  Done --> Next{User
choice}:::decision
  Next -- Scan another --> ScanLanding
  Next -- Back to asset --> Chooser

Backend permissions required for the field-tech role:
field-scan:access · maintenance-reports:create · maintenance-reports:read · maintenance-reports:sign · assets:read-id · assets:read-sku · assets:check-sku · assets:bind-nfc-tag
